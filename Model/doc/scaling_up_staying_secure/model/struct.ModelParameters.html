Struct that defines the model parameters

Fields§

§name: String

Naming for the model. Used for logs etc.

§steps: Steps

Number of steps in the Model

§seed: SmallRng

Seed for the random generators, same behaviour between runs

§logfolder: String

Folder to store logs

§logfile: String

Chrono - format string to determine the filename. link

§enable_output: bool

Enable the output of the node states per step.

§type_output: String

Type of output, avro or json.

§outputdatafolder: String

Folder to store the output

§outputdatafile: String

Chrono - format string to determine the filename. link

§outputcodec: Codec

The codec that is used. Currently supported: “snappy”, “deflate”, ’null“, See also [Codec].

§attack: Attack

The attack struct of this model. See Attack.

§grid: Grid

The Grid struct of this model. See Grid.

§loglevel: LevelFilter

The log level of the executable.

§stop_on_freq_error: bool

Stop running the model when there is a grid frequency error.
