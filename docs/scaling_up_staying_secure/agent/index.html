Module scaling_up_staying_secure::agent

source ·
Expand description

Defines the agents that are part of the model and the trait that is used to create new ones.

To take part in the simulation agents need to implement the AgentTrait trait. This allows them to be a part of the underlying graph and take part in the grid modelling.

Furthermore, some types are defined here that make it easier to read the code. These are nothing else than other names for data structures.

New Agents

Implementing new agents can be done by implementing AgentTrait and adding the discriminant to the AgentKind enum. The behaviour during each step of the model needs to be defined in the model part.
